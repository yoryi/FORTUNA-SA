## FORTUNA SA
prueba tecnica

## Iniciar Proyecto

Instalar Proyecto

```
$ cd ../FORTUNA-SA
$ npm install
$ cd ios
$ pod install
```

Correr Proyecto

```
$ npm run ios
$ npm run android

```
